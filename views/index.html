Get Started

First of all we will deploy this to our heroku server. You are just one click from the deployment, you find a button button Deploy to Heroku.

Base URL

Once your app is deployed to the heroku then you are ready to try this api. You can find the App URL on heroku and it's going to be your base url.

Your base url will look like this https://your-app-name.herokuapp.com

Endpoint will be like this: base_url + path
